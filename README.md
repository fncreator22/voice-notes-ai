# Voice Notes AI

A speech-to-text application powered by Whisper AI.

## Features

* Record audio from microphone
* Upload audio files
* Convert speech to text
* Download transcript as TXT file

## Technology Stack

* Python
* Gradio
* Hugging Face Transformers
* Whisper Base

## Model

openai/whisper-base

## Run

pip install -r requirements.txt

python app.py
